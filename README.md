# apidev

Binary CLI tool for contract-driven API scaffolding.

## Commands

- `apidev init`
- `apidev validate`
- `apidev diff`
- `apidev gen`

Compatibility alias:
- `apidev generate` (legacy alias for `apidev gen`)
